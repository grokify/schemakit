# Installation

## Homebrew

```bash
brew install grokify/tap/schemakit
```

## Go Install

```bash
go install github.com/grokify/schemakit/cmd/schemakit@latest
```

## From Source

```bash
git clone https://github.com/grokify/schemakit.git
cd schemakit
go build -o schemakit ./cmd/schemakit
```

## Verify Installation

```bash
schemakit version
```
